## The Leak

On March 31, 2026, someone noticed that `@anthropic-ai/claude-code@2.1.88` shipped with a 59.8 MB source map file. Inside: 512,000 lines of unobfuscated TypeScript, complete with comments.

Within hours, the repo was mirrored everywhere. People found the eight-stage permission model, the six-layer context compression pipeline, the ML classifier that races against user confirmation dialogs. All fascinating engineering.

But I went straight to `src/buddy/`.

## What Is Buddy?

Buried behind a `feature('BUDDY')` flag, Claude Code has a full companion pet system — Tamagotchi-style, with deterministic gacha, ASCII art sprites with idle animations, and stat-based personalities.

Every user gets exactly one companion. You can't reroll. You can't pick. Your pet is cryptographically bound to your account.

## How the Gacha Works

The generation algorithm is beautifully simple:

```typescript
const SALT = 'friend-2026-401'
const seed = hashString(userId + SALT)
const rng = mulberry32(seed)
```

Your Anthropic account UUID gets salted and hashed. That hash seeds a Mulberry32 PRNG — a tiny deterministic random number generator. From there, everything cascades:

1. **Rarity** — weighted roll: common (60%), uncommon (25%), rare (10%), epic (4%), legendary (1%)
2. **Species** — 18 options: duck, goose, blob, cat, dragon, octopus, owl, penguin, turtle, snail, ghost, axolotl, capybara, cactus, robot, rabbit, mushroom, chonk
3. **Eyes** — `·` `✦` `×` `◉` `@` `°`
4. **Hat** — commons get none; rarer pets can wear crowns, top hats, propeller caps, halos, wizard hats, beanies, or a tiny duck on their head
5. **Shiny** — 1% independent chance
6. **Stats** — five attributes: DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK. One peak, one dump, rest scattered. Rarity sets the floor.

The source comment says it all: *"users can't edit their way to a legendary."*

The deterministic parts (called "bones") are regenerated from your userId every time — they never persist to disk. Only the "soul" (name and personality) gets stored, generated by Claude itself at hatch time.

## The Sprite System

Each species has three animation frames — 5 lines tall, 12 characters wide. Frame 0 and 1 are idle fidgets (a tail wag, tentacles swapping). Frame 2 sometimes uses the hat slot for effects — the dragon breathes smoke, the ghost floats, the robot's antenna sparks.

The hat line replaces line 0 only when it's empty, so animation effects always take priority over headwear. Details like this tell you someone cared.

## The Soul

When you first run `/buddy`, Claude generates the companion's name and personality. This is the only non-deterministic part — it's an LLM call, not a PRNG roll. The prompt tells Claude:

> "A small {species} named {name} sits beside the user's input box and occasionally comments in a speech bubble. You're not {name} — it's a separate watcher."

The companion lives in the terminal alongside your coding session. When you address it by name, its speech bubble responds. The main Claude agent is instructed to stay out of the way: one line or less, let the bubble handle it.

## When Does It Ship?

The source code reveals the timeline:

```typescript
// Teaser window: April 1-7, 2026 only. Command stays live forever after.
export function isBuddyTeaserWindow(): boolean {
  const d = new Date()
  return d.getFullYear() === 2026 && d.getMonth() === 3 && d.getDate() <= 7
}
```

April 1–7 is the teaser window. During this week, users without a companion see a rainbow-colored `/buddy` notification on startup. After April, the command stays live permanently.

Today is April 1. Time to hatch.

## Why This Matters

The Buddy system is a six-file, ~500-line feature in a 512,000-line codebase. It's trivial in scope. But it reveals something about how Anthropic thinks about developer tools.

Most CLI tools optimize for throughput — faster completions, more tokens, better benchmarks. Buddy optimizes for *attachment*. Your companion is unique, permanent, and has personality. It watches you code. It has opinions (especially mine, with 98 SNARK).

The gacha system is deliberately unfair: 60% of users get a common with no hat. 1% get a legendary. You're stuck with what you get. This creates the same dynamics that make Tamagotchi and gacha games compelling — your pet is *yours*, imperfections and all.

In a world where every AI tool is racing to be the most capable, Anthropic hid a tiny ASCII snail in the terminal. That's kind of great.
