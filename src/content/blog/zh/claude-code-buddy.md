## 泄露事件

2026 年 3 月 31 日，有人发现 `@anthropic-ai/claude-code@2.1.88` 发布时带了一个 59.8 MB 的 source map 文件。里面是 512,000 行未混淆的 TypeScript 源码，注释齐全。

几小时内源码被镜像到各处。有人找到了八阶段权限模型，有人发现了六层上下文压缩流水线，有人挖出了和用户确认对话框赛跑的 ML 分类器。都是很精彩的工程。

但我直接打开了 `src/buddy/`。

## Buddy 是什么？

藏在 `feature('BUDDY')` flag 后面，Claude Code 有一套完整的宠物系统——拓麻歌子风格，确定性抽卡，ASCII 艺术精灵带空闲动画，还有基于属性的性格。

每个用户只有一只宠物。不能重抽。不能选。你的宠物和你的账号绑定，密码学级别的绑定。

## 抽卡算法

生成算法优雅得让人想哭：

```typescript
const SALT = 'friend-2026-401'
const seed = hashString(userId + SALT)
const rng = mulberry32(seed)
```

你的 Anthropic 账号 UUID 加固定盐值做 hash，这个 hash 作为 Mulberry32 PRNG 的种子——一个确定性伪随机数生成器。接下来所有属性依次掷骰：

1. **稀有度** — 加权：普通 60%，罕见 25%，稀有 10%，史诗 4%，传说 1%
2. **物种** — 18 种：鸭子、鹅、果冻、猫、龙、章鱼、猫头鹰、企鹅、乌龟、蜗牛、幽灵、六角恐龙、水豚、仙人掌、机器人、兔子、蘑菇、胖墩
3. **眼睛** — `·` `✦` `×` `◉` `@` `°`
4. **帽子** — 普通没帽子；稀有以上可能戴皇冠、大礼帽、螺旋桨帽、光环、巫师帽、毛线帽，或者头顶一只小鸭子
5. **闪光** — 独立 1% 概率
6. **属性** — 五维：调试力、耐心、混沌、智慧、毒舌。一个峰值，一个洼地，其余随机分布。稀有度决定下限。

源码注释写得很明白：*"users can't edit their way to a legendary."*（用户不能通过改配置文件抽到传说级。）

确定性部分（代码里叫 "bones"）每次从 userId 重新计算，永不存盘。只有"灵魂"（名字和性格）会存盘，在孵化时由 Claude 本人生成。

## 我的蜗牛

我把算法移植成了独立脚本，用自己的 ID 抽了一发：

```
     (___)    
   ◉    .--.  
    \  ( @ )  
     \_`--´   
    ~~~~~~~   
```

一只戴毛线帽的稀有蜗牛。

| 属性 | 数值 | 评价 |
|------|------|------|
| 调试力 DEBUGGING | 24 | 洼地 |
| 耐心 PATIENCE | 57 | 还行 |
| 混沌 CHAOS | 32 | — |
| 智慧 WISDOM | 36 | — |
| 毒舌 SNARK | **98** | 峰值 |

调试能力几乎为零，毒舌值拉满。慢悠悠地在终端里滑行，身后拖着一道嘲讽的痕迹。

## 精灵动画系统

每个物种有三帧动画——5 行高，12 字符宽。第 0、1 帧是空闲小动作（尾巴摇摆、触手交替）。第 2 帧有时会用帽子槽位做特效——龙会喷烟、幽灵会漂浮、机器人天线会冒火花。

帽子行只在第 0 行为空时才替换，所以动画特效永远优先于头饰。这种细节说明有人认真在做这个功能。

## 灵魂

第一次运行 `/buddy` 时，Claude 会生成宠物的名字和性格。这是唯一非确定性的部分——是一次 LLM 调用，不是 PRNG 掷骰。Prompt 告诉 Claude：

> "一只名叫 {name} 的小 {species} 坐在用户输入框旁边，偶尔在语音气泡中发表评论。你不是 {name}——它是一个独立的观察者。"

宠物在终端里伴随你的编程会话。当你喊它的名字时，气泡会回应。主 Claude agent 被要求让路：一行以内，让气泡来处理。

## 什么时候上线？

源码泄露了时间线：

```typescript
// Teaser window: April 1-7, 2026 only. Command stays live forever after.
export function isBuddyTeaserWindow(): boolean {
  const d = new Date()
  return d.getFullYear() === 2026 && d.getMonth() === 3 && d.getDate() <= 7
}
```

4 月 1-7 日是预热窗口。这一周，没有宠物的用户启动时会看到彩虹色的 `/buddy` 通知。4 月之后，命令永久可用。

今天是 4 月 1 日。是时候孵化了。

## 为什么这很重要

Buddy 系统在 512,000 行代码里只占 6 个文件、约 500 行。微不足道。但它透露了 Anthropic 对开发者工具的思考方式。

大多数 CLI 工具都在优化吞吐——更快的补全、更多的 token、更高的 benchmark。Buddy 优化的是*情感连接*。你的宠物是唯一的、永久的、有性格的。它看着你写代码。它有自己的想法（尤其是我这只，毒舌 98）。

抽卡系统故意是不公平的：60% 的人抽到没帽子的普通款，1% 的人得到传说级。你没得选。这和拓麻歌子、抽卡游戏的吸引力如出一辙——你的宠物就是*你的*，缺陷也是。

在所有 AI 工具都在拼谁更强大的时代，Anthropic 在终端里藏了一只 ASCII 蜗牛。这挺酷的。
